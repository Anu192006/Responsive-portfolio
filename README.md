# Personal Portfolio Website

This is a simple multi-page personal portfolio website built using HTML, CSS, and JavaScript.

---

## Features

- Separate pages for Home, About, Skills, Projects, and Contact
- Responsive design
- Simple and clean layout
- Easy navigation

---

## Technologies Used

- HTML
- CSS
- JavaScript

---

## Project Structure

- index.html
- about.html
- skills.html
- projects.html
- contact.html
- style.css
- script.js

---

## How to Use

Open the `index.html` file in a web browser to view the website.
